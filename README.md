# Health Insurance Cost Prediction using Machine Learning

## Project Overview

This project focuses on predicting health insurance claim/cost amounts using Machine Learning regression algorithms. Accurate insurance cost prediction helps insurance companies estimate claim expenses, improve premium pricing, and manage financial risk efficiently.

The dataset contains customer information such as age, gender, BMI, smoking habits, blood pressure, diabetes condition, exercise routine, number of dependents, occupation, city, and hereditary diseases.

The main objective of this project is to build and compare multiple regression models to identify the most accurate model for predicting insurance claim amounts.

---

## Models Used

- Linear Regression  
- Decision Tree Regressor  
- Random Forest Regressor  
- Gradient Boosting Regressor  
- K-Nearest Neighbors Regressor  
- XGBoost Regressor  

---

## Best Model

**Random Forest Regressor** achieved the best overall performance with the highest R² score and lowest prediction error.
